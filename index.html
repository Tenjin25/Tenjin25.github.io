NC Political Realignment Map (2008-2024) (Featured Project)

NC Political Realignment Map (2008-2024) is a comprehensive, interactive web application for visualizing North Carolina’s county and precinct-level election results and trends. It leverages Mapbox GL JS for map rendering and Python scripts for data processing and normalization.

  • Interactive county and precinct maps with color-coded partisan margins
  • Sidebar with detailed county analysis and research findings
  • Dynamic swing arrow visualizations for election comparisons
  • Robust backend data merging and normalization
  • Responsive UI with customizable controls and real-time status feedback
Latest Updates from August 24:
  • Instant County Zoom: Instantly zoom to any county by typing its name in the sidebar search box for fast navigation and comparison.
  • UI & Readability Upgrades: Ballotpedia-standard R+/D+ margins, clearer statewide results, improved tossup clarity, and standardized formatting for a more professional look.
  • Behind-the-Scenes Improvements: Codebase cleanup for reliability, better error handling, and manual data overrides for enhanced accuracy.

As the lead developer, I was responsible for the full-stack architecture, UI/UX design, and data integration. The project demonstrates advanced skills in web mapping, data visualization, and modern JavaScript frameworks.

This project is actively maintained and improved with new features and updates as needed. Development began in July 2025.
An enhanced version with interactive precincts is in development and planned for release by the end of 2025. This is the latest stable version.

  • Key Features: Interactive maps, demographic filters, election data visualization, custom district drawing, and export options.
  • Main resources used: Mapbox GL JS, Turf.js, scripts/common.js, styles/common.css, preview.png
  • Role: Primary developer and designer (90% of the work), with GitHub Copilot as an aid—not a crutch. Responsible for full-stack architecture, UI/UX, and data integration.

Data sources: Official election data from NCSBE, county shapefiles from NCOneMap (NCDOT), and Census VTDs from the U.S. Census Bureau.
County data was cross-referenced with Ballotpedia for accuracy to ensure correct data merging. The same formula and decimal display as Ballotpedia was used for a professional look.

What do the Color-Coded Categories Mean?
  • Annihilation (40%+ margin): Landslide victory, safe stronghold.
  • Dominant (30-40% margin): Very safe seat, slightly less extreme than Annihilation.
  • Stronghold (20-30% margin): Reliably safe for the winning party.
  • Safe (10-20% margin): Considered safe, but not impenetrable.
  • Likely (5.5-10% margin): Clear advantage, but could become competitive.
  • Lean (1-5.5% margin): Competitive, modest advantage for the winner.
  • Tilt (0.5-1% margin): Extremely competitive, slight edge for the winner.
  • Tossup (±0.5% margin): True battleground, no clear favorite.

These categories help users quickly identify which areas are safe for each party, which are competitive, and where political realignment is occurring.

Florida Realignment (Featured Project)

Florida Realignment is an interactive mapping tool for visualizing Florida's political landscape and the impact of redistricting. Users can explore the 2024 presidential election results using my custom rating system, analyze county-level data, and view dynamic, color-coded maps.

Work in progress: This project is under active development and new features are being added regularly.

Margin of Victory

Margin of Victory is a data analysis project that visualizes and compares the margin of victory in U.S. presidential elections at the county level. It started as a simple Java calculator app, which I modified to calculate the winner's margin of victory and assign a rating based on the output, electoral competitiveness, and trends.

  • Margin of Victory calculation and professional race rating system (Tossup, Tilt, Lean, Likely, Safe).
  • Winner and party determination, flip detection, and robust input validation.
  • Real-world case studies and split-ticket voting analysis included in documentation.

This project inspired the creation of NC Political Realignment Map and Florida Realignment.
